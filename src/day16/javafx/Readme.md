### JAVA FX 세팅

- OPEN JDK 19 설치
```
[window] -> [preperences] -> [Java] -> [Installed JREs.. ]
	->[ADD] -> "OPEN JDK19" 폴더 선택
```

- JavaFX SDK 다운로드
https://gluonhq.com/products/javafx/

- 이클립스 javafx 플러그인 e(fx)clipse 설치
```
[Help] -> [Eclipse Marketplace] -> "javafx" 검색 -> e(fx)clipse install
```

- SceneBuilder 다운로드
https://gluonhq.com/products/scene-builder/


- ScenBuilder 이클립스 연동
```
[Windows] -> [Preferences] -> [JavaFX] -> [SceneBuilder executable]
	-> SceneBuilder.exe 
```
